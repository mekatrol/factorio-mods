# Tree Spawn Mod Commands

## Hotkeys

| Shortcut | Custom input | Action |
| --- | --- | --- |
| `Ctrl + T` | `augment-toggle-tree` | Toggle periodic tree spawning near the player. |

## Behavior

When enabled, the mod periodically tries to place a tree near each connected
player that has tree spawning enabled.

## Console Commands

This mod does not register console commands.
