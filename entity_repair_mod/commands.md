# Entity Repair Mod Commands

## Hotkeys

| Shortcut | Custom input | Action |
| --- | --- | --- |
| `Ctrl + Shift + W` | `mekatrol-toggle-repair-bot` | Toggle the repair bot on or off. |

## Behavior

When enabled, the repair bot follows the player, tracks damaged or destroyed
entities, uses repair pack durability, and clears its visual/pathfinding state
when disabled.

## Console Commands

This mod does not register console commands.
