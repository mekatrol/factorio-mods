# Mapping Bot Mod Commands

## Hotkeys

| Shortcut | Custom input | Action |
| --- | --- | --- |
| `Ctrl + Shift + M` | `mekatrol-mapping-bot-toggle` | Toggle the mapping bot on or off. |
| `Ctrl + Shift + N` | `mekatrol-mapping-bot-clear` | Clear mapped entities and their visuals. |

## Behavior

When enabled, the mapping bot scans nearby static entities, records mapped
entities in shared mod storage, draws mapping visuals, and raises generated
custom events for mapping and clear actions.

## Console Commands

This mod does not register console commands.
